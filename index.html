AI Document Reading — API Key
Not set
PropTrack can read your receipts, invoices, leases and tax bills automatically using AI. Drop a file in Upload Docs and it fills in all the fields for you.

To enable this, you need a free Anthropic API key:
Step 1 — Create your free Anthropic account
Go to console.anthropic.com and sign up. It's free to create an account.
Step 2 — Add a small credit balance
In the console go to Billing and add $5. Reading one document costs less than $0.01, so $5 covers thousands of documents.
Step 3 — Create an API key
In the console click API Keys in the left sidebar → click Create Key → give it any name → copy the key that starts with sk-ant-
Step 4 — Paste your key here
Key saved — AI reading is active! Go to Upload Docs to try it.
Your key is stored only in this browser's local storage. It never leaves your computer except to go directly to Anthropic when you upload a document.
